# Phase 2 - Zero-Cost Agent Runtime

## Goal

Build a functional agentic runtime without depending on LLMs or paid APIs.

## Principles

- No LLM dependency.
- Local-first.
- Rule-based where possible.
- Inspectable decisions.
- Domain knowledge as structured data.
- Templates before generation.
- Optional LLM adapter later, never mandatory.

## Planned components

1. Intent Matcher
2. Entity Extractor
3. Template Response Engine
4. Dialogue State
5. Goal Tracker
6. Local Knowledge Search
7. Fallback Engine
