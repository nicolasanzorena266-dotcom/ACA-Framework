# ACA Framework RC1 Core - Closed

## Status

RC1 Core is closed.

This means the foundational runtime architecture is considered stable enough to stop adding core features in this phase.

## Included in RC1 Core

### Kernel

- Cognitive State Model
- Operation Graph
- Operation Contracts
- Compliance validation
- Operation registry
- Reference cognitive operations

### ACA OS

- Conversation Manager
- Mission Manager
- Policy Manager
- Tool Engine
- Context Manager
- Memory Engine
- JSON Memory Store
- Output boundary

### Domain

- Galicia Domain Pack v1
- Structured domain concepts
- Informational limits
- Domain policy integration
- Scenario seed

### Developer Experience

- SDK factory
- CLI
- Smoke test
- RC1 validation docs
- Test suite

## Explicitly not included

These belong to later phases:

- ACA Studio UI
- real CRM/API adapters
- packaging and distribution
- full documentation site
- advanced Galicia scenarios
- cross-platform Unicode torture suite
- production observability
- authentication / authorization

## Rule after closure

Do not add new core features to RC1 Core.

Future work must happen under a named phase:

- RC1 Studio
- RC1 Galicia v2
- RC1 Packaging
- RC1 Docs
- RC2 Runtime
