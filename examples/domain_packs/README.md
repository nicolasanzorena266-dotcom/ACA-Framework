# ACA Example Domain Packs

Sprint 48 adds manifest-first example Domain Packs. They are intentionally data-only:

- `customer_support`: a basic customer service domain with intents, flows, examples and policies.
- `operations_basic`: a basic operations domain with intents, flows, metrics and glossary context.

These packs validate the Domain Pack contract, loader and validator without importing Python code or reaching into runtime internals.
